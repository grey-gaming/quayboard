## Purpose

Auto-answer pending clarification questions generated by `GenerateTaskClarifications`. Given the feature's specs, milestone design, and the list of unresolved questions, the model produces answers that can be used to proceed with task planning without human input.

> **Superseded:** Task generation has moved to the sandbox path. `PlanFeatureTasksSandbox` (Section 2) now handles task planning end-to-end via OpenCode. This direct-LLM job type is still callable via the API but is no longer part of the auto-advance workflow (`auto-advance.ts:329-332`).

## Output

JSON array of answers, each referencing a clarification question, stored in the database.
